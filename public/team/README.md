# Dossier Photos d'Équipe

## Instructions

Déposez votre photo ici avec le nom : **`lydia.jpg`**

### Recommandations :
- **Format** : JPG ou PNG
- **Dimensions recommandées** : 1200x900 pixels (ratio 4:3)
- **Taille maximale** : 2 MB
- **Qualité** : Haute résolution, bien éclairée

### Votre photo apparaîtra sur la page "À Propos"

Une fois la photo déposée, elle sera automatiquement affichée avec :
- Votre nom : "Lydia"
- Votre titre : "Fondatrice & Décoratrice"
- Un dégradé élégant en bas de l'image
